Mashit

Boris Vandenbergh

Maak een rich internet application in php & javascript waarbij je APIs kunt doorzoeken. De zoekresultaten kun je als gebruiker verder filteren (relevantie van een resultaat aanduiden, of niet-relevante zaken wissen uit de resultaten). De persoonlijke resultaten kan de gebruiker opslaan als een project. Een gebruiker kan meer dan 1 project (met persoonlijk gefilterde resultaten) hebben.
